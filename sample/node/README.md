## Introduction

This project demonstrates how to use the `ti-js` library in a node server.

## Setup

Run:
```sh
npm install
```

## Usage

To start the server, run:
```sh
npm start
```

Then navigate to `http://127.0.0.1:3000/` in your favorite web browser.
